Guía rápida para jugadores

¡Compite, adivina y suma puntos!

Dos equipos se enfrentan para acertar las respuestas más populares de cada pregunta. El equipo que mejor coordine y acumule más puntos, gana.

Objetivo

Dos equipos compiten para adivinar las respuestas más populares de cada pregunta. Cada respuesta suma puntos y gana el equipo que alcance la meta o cierre con mejor puntaje.

Cómo se juega una ronda

  1. Se muestra una pregunta: ambos equipos escuchan y se preparan.
  2. Un equipo toma el turno: ese equipo responde primero.
  3. Si acierta: aparece la respuesta y suma puntos.
  4. Si falla: recibe un strike (máximo 3).
  5. Cierre de ronda: se entregan puntos al equipo que corresponde según el resultado.

Puntaje

Las respuestas tienen valores distintos: entre más popular, más puntos.

Puede haber rondas con multiplicador (x2 o x3), así que cada acierto vale más.

Si un equipo llega a 3 strikes, el rival puede intentar quedarse con los puntos de la ronda.

La partida termina cuando se alcanza la meta de puntos o cuando se cierra manualmente para declarar ganador.

Recomendaciones para jugadores

Escucha bien la pregunta: evita responder por impulso.

Hablen corto y claro: acuerden una respuesta rápida en equipo.

Cuida los strikes: cada error acerca la posibilidad de robo del rival.

Disfruten el ritmo: el juego es más divertido cuando todos participan.
